# Lexicon — Personal Dictionary App

A full stack dictionary web app built with React, Node.js, Express, and MongoDB.

## Features
- Search any English word with real definitions
- Phonetic pronunciation with audio playback
- Autocomplete suggestions while typing
- Personal account — your saved words are private
- Add and delete custom words

## Tech Stack
- Frontend: React + Vite
- Backend: Node.js + Express
- Database: MongoDB Atlas
- Auth: JWT + bcrypt
- Dictionary API: Free Dictionary API + Datamuse API

## Run Locally

cd backend && npm install && npm run dev
cd frontend && npm install && npm run dev
